# TO DO APPLICATION 

this app allows users to track their tasks for a day.

#### Instruction
1. Download the source code

```js
git clone [https://github.com/unicage/webCW.git]
```

or just download the zip file 

2. Install deps
```js
npm install
```

3. Run the application
```js
node app
```

#### Deps list
- express
- pug
